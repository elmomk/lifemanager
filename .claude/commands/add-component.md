Create a new reusable Dioxus component.

Arguments: $ARGUMENTS (component name and description)

Steps:
1. Read `src/components/mod.rs` and an existing component for patterns
2. Create the new component file in `src/components/`
3. Register it in `src/components/mod.rs`
4. Follow the project's glassmorphism design language with dark mode support
5. Run `cargo check` to verify it compiles
